# Tuby V1.0

GUI that allows you to download MP3/MP4 files through Youtube
![Image](https://i.ibb.co/mc6WdHK/Annotation-2020-07-02-230341.png)

## Installation

Download and run Tuby.exe

## Usage
1) Paste Youtube video URL 
2) Enter File Name
3) Select destination 
4) Download!
