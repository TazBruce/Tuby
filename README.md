# Tuby V1.0

GUI that allows you to download MP4 files through Youtube
![Image](https://i.ibb.co/BgZ9cHN/Annotation-2020-07-02-230341.png)

## Installation

Download and run Tuby.exe

## Usage
1) Paste Youtube video URL 
2) Select destination 
3) Download!
